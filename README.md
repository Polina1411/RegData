##  Ключевые особенности

**RegData** — это интерактивная аналитическая платформа, разработанная для исследования взаимосвязей между экономической свободой, регуляторной средой и ключевыми макроэкономическими показателями стран мира. Проект объединяет данные из авторитетных международных источников и предоставляет инструменты для глубокого сравнительного анализа.

**Цель проекта:**
- Визуализация комплексных экономических данных в интуитивно понятном формате
- Выявление корреляций и тенденций между регуляторной политикой и экономическими результатами
- Создание инструмента для исследователей, политиков и студентов экономических специальностей
- Демонстрация практического применения данных для принятия обоснованных решений

##  Источники данных

### Основные источники
1. **The Heritage Foundation** — Индекс экономической свободы
   - Общий индекс и 12 компонентов
   - Данные с 1995 года по настоящее время
   - 186 стран

2. **Всемирный банк** — World Development Indicators
   - ВВП, ВВП на душу населения
   - Инфляция, безработица
   - Прямые иностранные инвестиции
   - Уровень бедности

3. **ОЭСР** — База экономических данных
   - Структурные экономические показатели
   - Данные по странам-членам ОЭСР

4. **Дополнительные источники**
   - Freedom House (политические свободы)
   - Transparency International (индекс коррупции)
   - МВФ (фискальные показатели)

### Методология интеграции данных
- Автоматическое обновление через API
- Согласование разных систем классификации стран
- Интерполяция пропущенных значений
- Нормализация шкал измерений

##  Функциональные возможности

### 1.  Интерактивная мировая карта
- **Цветовая кодировка** по выбранному показателю
- **Многоуровневое масштабирование** (континенты → страны → регионы)
- **Информационные всплывающие окна** при наведении
- **Динамическое обновление** при выборе года/показателя
- **Экспорт изображений** в высоком разрешении

### 2.  Панель детального анализа страны
- **Радар-диаграмма** компонентов экономической свободы:
  - Права собственности
  - Эффективность судебной системы
  - Добросовестность правительства
  - Налоговое бремя
  - Государственные расходы
  - Фискальное здоровье
  - Свобода бизнеса
  - Свобода труда
  - Денежная свобода
  - Свобода торговли
  - Свобода инвестиций
  - Финансовая свобода

- **Ключевые показатели:**
  - Общий индекс экономической свободы
  - Мировой рейтинг
  - Региональный рейтинг
  - Динамика изменений за 10 лет

### 3.  Корреляционный анализ
- **Диаграммы рассеяния** с линией тренда
- **Расчет коэффициента корреляции Пирсона**
- **Визуализация доверительных интервалов**
- **Фильтрация по группам стран** (развитые/развивающиеся)
- **Многомерный анализ** (3D-графики)

### 4.  Сравнительный анализ
- **Выбор до 5 стран** для сравнения
- **Параллельные координаты** для многомерного сравнения
- **Интерактивные гистограммы**
- **Таблицы сравнительных показателей**
- **Автоматическое выделение лидеров/аутсайдеров**

### 5.  Анализ временных рядов
- **Графики динамики** за выбранный период
- **Скользящие средние** и трендовые линии
- **Прогнозирование** на основе исторических данных
- **Выявление структурных изменений**

### 6.  Пользовательские дашборды
- **Drag-and-drop интерфейс** для создания дашбордов
- **Предустановленные шаблоны** для разных исследовательских задач
- **Экспорт результатов** в PDF/PNG/CSV
- **Сохранение конфигураций** для последующего использования

##  Техническая реализация

### Модель вычисления корреляций
- **Методы:** Пирсон, Спирмен, Кендалл
- **Учет временных лагов** (1-5 лет)
- **Контроль ковариат** (уровень развития, регион)
- **Статистическая значимость** (p-value, доверительные интервалы)

### Оптимизация производительности
- **Инкрементальное обновление** данных
- **Кэширование** вычислений
- **Ленивая загрузка** графиков
- **Оптимизация запросов** к базе данных

##  Научная значимость

### Исследовательские вопросы
1. Как различные компоненты экономической свободы влияют на экономический рост?
2. Существует ли оптимальный уровень регулирования для разных типов экономик?
3. Как экономическая свобода коррелирует с социальными показателями?
4. Каков временной лаг между либерализацией и экономическими эффектами?

### Методологические особенности
- **Прозрачность расчетов** — все формулы доступны в документации
- **Воспроизводимость** — детальное описание методологии
- **Сравнительный подход** — анализ по группам стран
- **Динамический анализ** — изучение изменений во времени

### Практическое применение
- **Для исследователей:** Быстрая проверка гипотез
- **Для политиков:** Обоснование реформ фактическими данными
- **Для студентов:** Интерактивное изучение экономики
- **Для инвесторов:** Оценка инвестиционного климата
